Design Direction — Degree of Change

You want dark premium indigo/cyan. How much do we reshape the layout vs re-skin?

A

Reskin — same bones, new colors

Keep exact same layout grid, component positions, section order. Apply dark palette, glass surfaces, new typography hierarchy, and updated micro-animations. Safest, fastest.

B

Reskin + UX polish

Same structure, but fix UX pain points: scroll-aware sticky nav, better hero background integration, improved card hover states, smoother section transitions, responsive refinements. Recommended.

C

Restructure sections

Reorganize some sections — merge problem + how-it-works, redesign bento grid layout, different pricing card arrangement. Higher risk, bigger payoff.
